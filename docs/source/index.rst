关卡设计之书
===================================

本书以平易近人、最新和批判性的方式汇聚3D电子游戏的关卡设计知识。它适用于具备各类经验水准和游戏引擎使用经历的策划。

.. note::

   这本书正在大力建设中，尚未公开发布。书籍结构和内容可能会有很大变化。
